================================================================================

   Разработка информационной системы для проведения онлайн-тестирования
   с ролевой моделью доступа, элементами геймификации и поддержкой групп
   пользователей

================================================================================
О ПРОЕКТЕ
================================================================================

Информационная система для проведения онлайн-тестирования в учебных заведениях.
Поддерживает три роли пользователей: студент, преподаватель, администратор.

КЛЮЧЕВЫЕ ВОЗМОЖНОСТИ:
   - Ролевая модель доступа (студент, преподаватель, администратор)
   - Создание тестов с вопросами (одиночный и множественный выбор)
   - Геймификация: уровни, опыт (XP), достижения, рейтинг
   - Группы студентов и фильтрация тестов по группам
   - Ограничение попыток прохождения теста
   - Шифрование паролей и правильных ответов
   - Экспорт результатов в CSV
   - Адаптивный дизайн

================================================================================
ТЕХНОЛОГИИ
================================================================================

   Языки и технологии: PHP 8.3, MySQL, HTML5, CSS3, JavaScript
   Локальный сервер:   MAMP (Apache + MySQL + PHP)
   Среда разработки:   Visual Studio Code

================================================================================
УСТАНОВКА И ЗАПУСК
================================================================================

1. УСТАНОВИТЕ MAMP
   Скачайте и установите MAMP с официального сайта: https://www.mamp.info/

2. СКОПИРУЙТЕ ПРОЕКТ
   Скопируйте папку testing_system в:
   - Windows: C:\MAMP\htdocs\
   - macOS:   /Applications/MAMP/htdocs/

3. ЗАПУСТИТЕ MAMP
   Откройте MAMP и нажмите "Start Servers"
   Убедитесь, что Apache и MySQL горят зелёным

4. НАСТРОЙТЕ БАЗУ ДАННЫХ
   - Откройте phpMyAdmin: http://localhost/phpmyadmin
   - Создайте базу данных testing_system (кодировка: utf8mb4_unicode_ci)
   - Нажмите "Импорт" и выберите файл database/testing_system_dump.sql
   - Нажмите "Применить"

5. НАСТРОЙТЕ ПОДКЛЮЧЕНИЕ
   - Переименуйте config/database.example.php в config/database.php
   - Откройте файл и укажите:
     пароль: root (или оставьте пустым, если не работает)

6. ОТКРОЙТЕ САЙТ
   Перейдите в браузере по адресу:
   http://localhost/testing_system/

================================================================================
ВОЗМОЖНЫЕ ПРОБЛЕМЫ
================================================================================

1. ОШИБКА ПОДКЛЮЧЕНИЯ К БД
   - Проверьте, запущен ли MySQL в MAMP
   - Проверьте пароль в config/database.php
   - Попробуйте пароль: root, или оставьте пустым ('')

2. СТРАНИЦА НЕ ОТКРЫВАЕТСЯ
   - Проверьте порт в адресной строке (обычно 8888 или 80)
   - Если порт 8888, то адрес: http://localhost:8888/testing_system/
   - Проверьте, что папка лежит в htdocs

3. ПАРОЛИ НЕ ПОДХОДЯТ
   - Попробуйте пароль 123 для всех аккаунтов
   - Или создайте нового пользователя в phpMyAdmin
